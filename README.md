# Marklist_SQL_HTML
This is a Program on Printing Marklist using HTML, CSS, PHP and MySQL. 
Marks.sql is the Database. 
MarksEntry.html is the input for the marks. 
Marks.php is the Main Output. 
Connected.php is the Connector.
